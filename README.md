# Week 1 - C Programming Module

## 📌 Overview
This repository contains the solutions and notes for **Week 1 C Programming Module**. The module focuses on understanding looping constructs in C and implementing different star pattern programs using nested loops.

---

## 📚 Topics Covered

### Looping Statements
- While Loop
- For Loop
- Do-While Loop

### Pattern Printing Programs
- Square Pattern
- Hollow Square Pattern
- Right Triangle Pattern
- Inverted Triangle Pattern
- Pyramid Pattern
- Inverted Pyramid Pattern
- Half Diamond Pattern
- Diamond Pattern
